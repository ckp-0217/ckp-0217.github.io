Hardhat 测试代理合约

2022-07-13 1 min read # 代理升级

hardaht 测试

通过使用合约工厂就可以直接调用合约方法
例如

    获取合约
    使用载入或者部署
    然后直接调用方法就可以
    const baseERC20Test = await ethers.getContractFactory("ERC20");
    test = await baseERC20Test.deploy();
    await test.deployed();
    await test.name();

合约代理

    如果合约是一个代理合约 
    这个test没有代理的方法 
    但是调用要通过这个test地址来调用fallback()再转到对应impl的方法
    
    同时hardhat会有一层abi校验
    如果这个合约没有对应的方法就会报错
    这里可以通过载入的方式来进行调用

    const ERC20Impl = await (await hre.ethers.getContractFactory("ERC20Impl").connect(owner).attach(test.address);
