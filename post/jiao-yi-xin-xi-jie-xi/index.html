交易信息解析

2023-01-15 2 min read # 交易解析

交易信息

一个完整的交易请求和回执,包括通过hash可以获取一个交易的全部信息

解析思路

原生转账交易

对于链的原生币的转账，只要看交易信息中的value和input data 就可以知道

获取交易信息

通过访问区块链客户端根据hash或者区块号获取详细信息
geth获取交易信息

  Web3j client = Web3j
                .build(new HttpService(""));
        String transactionHash = "0xdc2d9370fd2fa28563a0ad0e38483a2d1eb15f28add78619871a7c0888a95221"; // 交易的 hash
        EthTransaction transaction = client.ethGetTransactionByHash(transactionHash).send();
        Transaction result = transaction.getResult(); // 获取交易信息
        //原生没有tostring方法可以使用反射
        Class<?> cls = result.getClass();
        Field[] fields = cls.getDeclaredFields();
        for (Field field : fields) {
            field.setAccessible(true);
            System.out.println(field.getName() + ": " + field.get(result));
        }

关键信息

hash:交易哈希

from:发送方
to:接收方
input data:发送参数
value:发送的ETH数量
logs:日志
status:状态

call:调用信息
