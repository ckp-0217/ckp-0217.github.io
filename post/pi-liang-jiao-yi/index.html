批量交易

发送和测试交易中,会遇到批量转账或者mint,执行批量转账或者mint时，就要通过脚本使用批量交易，使用默认的方式会使用同一个nonce，最终导致最多一笔交易上链，其他交易失败。

for (let index = 0; index < params; index++) {
    tokenA.connect(Wallet).mint(hre.ethers.utils.parseEther('0.01'), MyAddress);
}

解决方案

脚本方案
1.针对每一笔交易都使用await 等待上一笔交易结束

for (let index = 0; index < params; index++) {
    await tokenA.connect(Wallet).mint(hre.ethers.utils.parseEther('0.01'), MyAddress);}

2.提前获取nonce，每一笔交易后把nonce递增。

let nonceNumber= await hre.ethers.provider.getTransactionCount(Wallet.address);
for (let index = 0; index < params; index++) {
    tokenA.connect(Wallet).mint(hre.ethers.utils.parseEther('0.01'), MyAddress,{nonce:nonceNumber});
    nonceNumber++;
}

3.合约方案
通过自定义合约再通过调用合约来进行交易

contract Batch{
    function batch(uint xxx,address token,address user,uint amount) public{
        for (uint index = 0; index < xxx; ++index) {
            IERC20(y).mint(user,amount);
        }
    }
}
